Uma aplicação para controle de materiais para determinadas tarefas.
O usuário pode cadastrar um projeto (Ex: construção de uma casa.) definindo as tarefas que serão desempanhadas nesse
projeto (Ex.:Contrução do alicerce.) com seu tempo de duração e então definindo os materiais usados em cada tarefa (Ex.: Cimento). 
O programa utiliza um banco de dados sql.
O banco precisa ser configurado com o usuario: root, e a senha: root.
Existe um arquivo com o nome controlemateriais.sql que pode ser usado para criar o banco para aplicacao.
